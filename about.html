Sobre o Projeto

Este projeto é um assistente de IA projetado para otimizar e automatizar tarefas de gestão de contratos.

Utilizando a API do Gemini, ele é capaz de analisar documentos, extrair informações, padronizar dados e realizar validações complexas, reduzindo significativamente o trabalho manual e o risco de erros.
